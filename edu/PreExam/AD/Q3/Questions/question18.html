Question 18

Which statement is correct regarding the use of application control for inspecting web applications?





Explanation:

When Application Control blocks a web application, it does so silently without displaying a replacement message to the user. This is different from web filtering, which can present a block message.

  • A. Application Control can indeed differentiate between applications but does not generally allow separate actions on child and parent applications.
  • B. Application Control signatures are separate from the Antivirus engine.
  • D. Application Control typically requires SSL Inspection to identify applications within HTTPS traffic.

References:

  • Fortinet Documentation - Application Control Overview
  • Fortinet Documentation - FortiOS 6.2.16 Cookbook
  • Fortinet Reddit - Application Control Explanation

Correct Answer:
A. Application control can identify child and parent applications, and perform different actions on them

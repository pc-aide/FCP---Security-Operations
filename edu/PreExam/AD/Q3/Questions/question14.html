Question 14

There are multiple dial-up IPsec VPNs configured in aggressive mode on the HQ FortiGate. The requirement is to connect dial-up users to their respective department VPN tunnels.

Which phase 1 setting you can configure to match the user to the tunnel?





Explanation:

In FortiGate IPsec VPN configurations, particularly in aggressive mode, the Peer ID can be used to match incoming connections to a specific VPN tunnel based on the identity of the connecting user. This allows the FortiGate to differentiate between users from different departments and assign them to their respective VPN tunnels.

References:

  • Fortinet Technical Tip - How to use Peer IDs
  • Fortinet Documentation - Peers and Authentication Groups
  • Reddit - Branch Office FGs are coming into core on wrong VPN tunnel

Correct Answer:
A. Peer ID

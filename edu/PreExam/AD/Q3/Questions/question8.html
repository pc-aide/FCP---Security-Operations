Which two statements explain antivirus scanning modes? (Choose two.)





Explanation:

Flow-based inspection mode (A): FortiGate performs antivirus scanning while simultaneously forwarding data to the client, resulting in lower latency. The file is buffered as itâ€™s scanned, allowing transmission to continue, unless a threat is detected.

Proxy-based inspection mode (D): This mode buffers the entire file before forwarding, which provides a deeper scan but may introduce higher latency. If the file is clean, it is then sent to the client.

Option B and C are incorrect because, in both modes, files larger than the buffer limit might not be fully scanned if they exceed buffer limitations.

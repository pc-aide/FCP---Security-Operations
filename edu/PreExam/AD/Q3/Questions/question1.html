Question 1

Which three methods are used by the collector agent for AD polling? (Choose three.)






Explanation:

A, B, and C are correct. Fortigate 7.4 Administrator Study Guide page 130 explains:

  • WinSecLog: This method involves reading the Windows Security Event Log to gather information about user logins and logouts.
  • WMI (Windows Management Instrumentation): This method allows the collector agent to query the AD for user login information using WMI queries.
  • NetAPI: This method uses the Windows Network API to retrieve user login information from the AD.

References:

  • Fortinet Community - Technical Tip: FSSO Agent in polling mode

Correct Answers:
A. WinSecLog
B. WMI
C. NetAPI

Question 17

Which two statements are correct when FortiGate enters conserve mode? (Choose two.)





Explanation:

According to the FortiGate 7.4 Study Guide (p. 460), FortiGate does not accept configuration changes in conserve mode, as it may increase memory usage further. It explicitly does NOT run any quarantine actions. You can configure IPS fail-open to control IPS behavior when the IPS socket buffer is full.

Correct Answer:
B. FortiGate refuses to accept configuration changes
D. FortiGate continues to transmit packets without IPS inspection when the fail-open global setting in IPS is enabled

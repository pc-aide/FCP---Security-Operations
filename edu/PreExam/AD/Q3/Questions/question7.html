Question 7

Refer to the exhibits.

The exhibits show a diagram of a FortiGate device connected to the network, and the firewall configuration. An administrator created a Deny policy with default settings to deny Webserver access for Remote-User2. The policy should work such that Remote-User1 must be able to access the Webserver while preventing Remote-User2 from accessing the Webserver. Which two configuration changes can the administrator make to the policy to deny Webserver access for Remote-User2? (Choose two.)





Explanation:

A and B are correct. According to page 64 of the Study Guide 7.4, enabling match-vip in the Deny policy and setting the Destination address as Webserver in the Deny policy will help to deny Webserver access for Remote-User2. However, match-vip is enabled by default after FortiOS 7.2.3, making A a bit confusing in the context of FortiOS 7.4 questions.

References:

  • Fortinet Documentation - Configuring a Virtual IP
  • Fortinet User Guide - FortiPortal 7.4.1
  • Spiceworks Community - Accessing Internal Server Through FortiGate
  • Blog - Crucial Information for FortiGate Users

Correct Answers:
A. Enable match-vip in the Deny policy.
B. Set the Destination address as Webserver in the Deny policy.

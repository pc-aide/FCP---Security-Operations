Which two statements about equal-cost multi-path (ECMP) configuration on FortiGate are true? (Choose two.)





Explanation:

A and D are correct. Page 90 states that when SD-WAN is enabled, the v4-ecmp-mode parameter is hidden, and you control the ECMP algorithm with the load-balance-mode setting. When SD-WAN is disabled, the ECMP algorithm is configured via the CLI under config system settings.

Correct Answers:
A. If SD-WAN is enabled, you control the load balancing algorithm with the parameter load-balance-mode.

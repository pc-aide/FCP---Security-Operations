Question 5

Refer to the exhibits, which show the firewall policy and an antivirus profile configuration.

Why is the user unable to receive a block replacement message when downloading an infected file for the first time?





Explanation:

D is correct. According to the FortiGate 7.4 Administrator Self-Paced Guide, page 200, in flow-based scanning, if a virus is detected, the final packet is dropped, making the file unusable to the end user. FortiGate caches the URL of the file, and if the user attempts to download again, rather than scanning the file again, the IPS engine sends a block message to the user.

References:

  • Fortinet Documentation - Inspecting Traffic Content Using Flow-Based Inspection

Correct Answer:
D. Flow-based inspection is used, which resets the last packet to the user.

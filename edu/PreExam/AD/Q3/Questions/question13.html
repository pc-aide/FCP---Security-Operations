Question 13

Refer to exhibit.
An administrator configured the web filtering profile shown in the exhibit to block access to all social networking sites except Twitter. However, when users try to access twitter.com, they are redirected to a FortiGuard web filtering block page.
Based on the exhibit, which configuration change can the administrator make to allow Twitter while blocking all other social networking sites?





Explanation:

To allow access to Twitter while blocking all other social networking sites, the administrator should set the action for Twitter in the Static URL Filter to Exempt. This configuration ensures that requests to twitter.com are not blocked, allowing users to access the site without redirection to the FortiGuard web filtering block page.

Details on Other Options:

  • A. On the Static URL Filter configuration set Type to Simple: Changing the type to Simple does not directly impact the blocking or allowing of specific sites.
  • B. On the FortiGuard Category Based Filter configuration set Action to Warning for Social Networking: Setting the action to Warning would still block access, but only notify users instead of providing full access to Twitter.
  • C. On the Static URL Filter configuration set Action to Monitor: This would only log access attempts but not allow them, still leading to a block page for Twitter.

Correct Answer:
D. On the Static URL Filter configuration set Action to Exempt

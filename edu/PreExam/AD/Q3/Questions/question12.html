Question 12

A FortiGate firewall policy is configured with active authentication; however, the user cannot authenticate when accessing a website.
Which protocol must FortiGate allow even though the user cannot authenticate?





Explanation:

When a user attempts to authenticate through active authentication, DNS resolution is essential for accessing websites, as it translates domain names into IP addresses. Even if the user has not yet authenticated, DNS traffic must be allowed for the user to reach the login page and initiate the authentication process.

Details on Other Options:

  • A. ICMP: ICMP is not required for web browsing or authentication and is primarily used for network diagnostics (e.g., ping).
  • C. DHCP: DHCP assigns IP addresses but is not directly involved in the authentication or web access process.
  • D. LDAP: LDAP may be used to retrieve user credentials for authentication, but itâ€™s not necessary for web traffic itself.

B. DNS

Question 13

An administrator configured a FortiGate to act as a collector for agentless polling mode. What must the administrator add to the FortiGate device to retrieve AD user group information?





Explanation:

Pag 126 FortiGate_7.4_Administrator - D is correct. When FortiGate is acting as a collector for agentless polling mode, you must select "Poll Active Directory Server" and configure the IP addresses and AD administrator credentials for each Domain Controller (DC). FortiGate uses LDAP to query Active Directory (AD) to retrieve user group information. Therefore, you must add the LDAP server to the Poll Active Directory Server configuration.

References:

  • Fortinet Documentation - Configuring an LDAP server
  • Fortinet Community - How to configure FortiGate to use an LDAP server

Correct Answer:
D. LDAP server

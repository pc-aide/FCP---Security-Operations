Question 4

What are two features of the NGFW profile-based mode? (Choose two.)





Explanation:

Policy-based mode is actually a new policy mode. You can add applications and web filtering categories directly to a policy without having to first create and configure application control or web filtering profiles

However, if NGFW mode is Policy-based, then the inspection mode for all policies in that VDOM is always flow and there is no option available in the policy to change it

References:

  • Fortinet Documentation - Profile-based NGFW vs Policy-based NGFW
  • Fortinet FortiGate Infrastructure Study Guide for FortiOS 7.2

Correct Answers:
C. NGFW profile-based mode policies support both flow inspection and proxy inspection.
D. NGFW profile-based mode supports applying applications and web filtering profiles in a firewall policy.

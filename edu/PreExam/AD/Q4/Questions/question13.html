Question 13

An administrator has configured a strict RPF check on FortiGate.

How does strict RPF check work?





Explanation:

Strict: In this mode, Fortigate also verifies that the matching route is the best route in the routing table. That is, if the route in table contains a matching route for the source address and the incoming interface, but there is a better route for the source address through another interface, then the RPF check fails.

Correct Answer:
A. Strict RPF checks the best route back to the source using the incoming interface.

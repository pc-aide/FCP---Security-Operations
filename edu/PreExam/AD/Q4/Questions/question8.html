Question 8

FortiGuard categories can be overridden and defined in different categories. To create a web rating override for the example.com home page, the override must be configured using a specific syntax.

Which two syntaxes are correct to configure a web rating override for the home page? (Choose two.)





Explanation:

B. www.example.com: This syntax specifies the full domain name, which is correct for overriding the web rating for the home page.

D. example.com: This is another valid syntax that represents the base domain and also applies to the home page.

C. www.example.com/index.hrml: This syntax includes a specific file path that does not match the intended home page (which typically would be the root or default file) and also contains a typo (.hrml instead of .html), making it incorrect.

A. www.example.com:443: While this specifies the port, it is not necessary for defining a web rating override for the home page and is not a standard practice for overriding in FortiGuard.

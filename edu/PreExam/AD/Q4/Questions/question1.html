Question 1

Which two attributes are required on a certificate so it can be used as a CA certificate on SSL inspection? (Choose two.)





Explanation:

B and D are correct. The CA=True value identifies the certificate as a CA certificate, and the keyUsage=keyCertSign value indicates that the certificate's corresponding private key is permitted to sign certificates. See RFC 5280 section 4.2.1.9 Basic Constraints.

Correct Answers:
B. The CA extension must be set to TRUE
D. The keyUsage extension must be set to keyCertSign

Question 4

Which two statements are true about the FGCP protocol? (Choose two.)





Explanation:

B: FGCP elects the primary FortiGate device.
D: FGCP runs only over the heartbeat links.

These statements are true because the FGCP protocol is specifically designed to operate over heartbeat links to ensure communication and synchronization between cluster units, and it is responsible for electing the primary FortiGate device within the cluster. The FortiGate Clustering Protocol (FGCP) is a proprietary HA solution whereby FortiGates can find other member FortiGates to negotiate and create a cluster.

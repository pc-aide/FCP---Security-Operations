Question 5

Refer to the exhibit which contains a RADIUS server configuration.

An administrator added a configuration for a new RADIUS server. While configuring, the administrator selected the Include in every user group option.

What is the impact of using the Include in every user group option in a RADIUS configuration?





Explanation:

When the "Include in every user group" option is selected for a RADIUS server in the FortiGate configuration, it effectively makes the users authenticated against that RADIUS server available to all user groups configured on the FortiGate device. This means that any user who can authenticate against that RADIUS server will be treated as part of every user group, allowing for broader access and simplifying group management for users authenticated via RADIUS.

Correct Answer:
A. This option places the RADIUS server, and all users who can authenticate against that server, into every FortiGate user group.

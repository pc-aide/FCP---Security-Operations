Question 14

A network administrator has enabled full SSL inspection and web filtering on FortiGate. When visiting any HTTPS websites, the browser reports certificate warning errors. When visiting HTTP websites, the browser does not report errors.

What is the reason for the certificate warning errors?





Explanation:

When a FortiGate firewall performs SSL inspection, it essentially acts as a "man-in-the-middle" by decrypting and re-encrypting HTTPS traffic. To avoid certificate warnings, the browser must trust the certificate that FortiGate uses to re-encrypt the traffic. If the FortiGate SSL inspection certificate is not installed as a trusted certificate in the browser or operating system, the browser will display certificate warnings.

Correct Answer:
B. The browser does not trust the certificate used by FortiGate for SSL inspection.

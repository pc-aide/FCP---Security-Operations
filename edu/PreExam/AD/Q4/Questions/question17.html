Question 17

Refer to the exhibit.

If option 5 is used with the IPS diagnostic command and the outcome is a decrease in the CPU usage, what is the correct conclusion?





Explanation:

If the CPU use decreases after that, it usually indicates that the volume of traffic being inspected is too high for that FortiGate model. If there are high-CPU use problems caused by the IPS, you can use the diagnose test application ipsmonitor command with option 5 to isolate where the problem might be. Option 5 enables IPS bypass mode. In this mode, the IPS engine is still running, but it is not inspecting traffic.

If the CPU use decreases after that, it usually indicates that the volume of traffic being inspected is too high for that FortiGate model. If the CPU use remains high after enabling IPS bypass mode, it usually indicates a problem in the IPS engine, which you must report to Fortinet Support.

Correct Answer:
B. The IPS engine is inspecting a high volume of traffic.

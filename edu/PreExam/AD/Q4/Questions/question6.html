Question 6

Which statement about the deployment of the Security Fabric in a multi-VDOM environment is true?





Explanation:

In a multi-VDOM environment, the Security Fabric topology typically only displays VDOMs that have active connections and devices. VDOMs without any connected devices will not be shown in the topology view, making option C true.

Correct Answer:
C. VDOMs without ports with connected devices are not displayed in the topology.

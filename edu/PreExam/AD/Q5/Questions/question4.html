Question 4

Which statement correctly describes NetAPI polling mode for the FSSO collector agent?





Explanation:

In NetAPI polling mode, the FSSO collector agent uses the Windows NetAPI to poll domain controllers (DCs) and gather information on user logins. This method does not require searching the Windows application event logs and does not specifically track logouts using the NetSessionEnum function. However, B is also partly correct because NetAPI polling can indeed increase bandwidth usage in larger networks, but this is a secondary effect rather than a core functional description.

The polling method scans a Microsoft or Windows DC using the windows API (netapi32.dll) for logged in users.

References:

  • Fortinet FortiGate Infrastructure Study Guide for FortiOS 7.2
  • Technical Note - FSSO NetAPI Polling Bandwidth Usage Calculator

Correct Answer:
D. The collector agent uses a Windows API to query DCs for user logins.

Question 7

Which method allows management access to the FortiGate CLI without network connectivity?




Explanation:

The serial console on a FortiGate allows you to connect directly to the device's terminal without going through the network, unlike protocols such as SSH or Telnet, which require a network connection. This is useful for initial configuration or in case of network failure, providing direct access to the CLI.

To use the serial console, you need a serial cable (e.g., an RJ45 to DB9 cable) and terminal software like PuTTY or Tera Term, configured with the correct baud rate (9600 bps by default for FortiGate).

References:

  • Connecting to the CLI - Fortinet Documentation
  • How to connect to the FortiGate and FortiAP - Fortinet Community

Correct Answer:
B. Serial console

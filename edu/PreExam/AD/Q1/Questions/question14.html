Question 14

Refer to the exhibit, which shows the IPS sensor configuration.

If traffic matches this IPS sensor, which two actions is the sensor expected to take? (Choose two.)




Explanation:

C. The sensor will allow attackers matching the Microsoft.Windows.iSCSI.Target.DoS signature: This is correct because the action for this signature is set to "Monitor," which does not block the traffic but only logs it.

D. The sensor will block all attacks aimed at Windows servers: This is also correct because the action for the "Windows" filter is set to "Block," which will block all traffic matching Windows-related attacks.

When the IPS engine compares traffic with the signatures in each filter, order matters. The Rules are similar to firewall policy matching; the engine evaluates the filters and signatures at the top of the list first, and applies the first match. The engine skips the subsequent filters.

FortiGate Security 7.2 StudyGuide p.392

Incorrect:

A. The sensor will gather a packet log for all matched traffic: This is incorrect because packet logging is only enabled for the Microsoft.Windows.iSCSI.Target.DoS signature, not for all matched traffic.

B. The sensor will reset all connections that match these signatures: This is incorrect because there is no indication of a "reset" action in the configuration; the actions are set to Monitor and Block.

References:

  • Microsoft iSCSI Target Server Documentation

Correct Answer:
C. The sensor will allow attackers matching the Microsoft.Windows.iSCSI.Target.DoS signature.
D. The sensor will block all attacks aimed at Windows servers.

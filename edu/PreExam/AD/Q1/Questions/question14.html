Question 14

Refer to the exhibit, which shows the IPS sensor configuration.
Exhibit Link

If traffic matches this IPS sensor, which two actions is the sensor expected to take? (Choose two.)




Explanation:

The IPS sensor is configured to generate logs for all matched traffic (A). However, due to the order of the signatures, it will allow traffic matching the Microsoft.Windows.iSCSI.Target.DoS signature (C). Other attacks aimed at Windows servers would be blocked, but the specific DoS signature is allowed, making option D incorrect.

References:

  • Microsoft iSCSI Target Server Documentation

Correct Answer:
A. The sensor will gather a packet log for all matched traffic.
C. The sensor will allow attackers matching the Microsoft.Windows.iSCSI.Target.DoS signature.

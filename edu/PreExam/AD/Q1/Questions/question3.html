Refer to the exhibit:

Which algorithm does SD-WAN use to distribute traffic that does not match any of the SD-WAN rules?




Explanation:

If traffic does not match any of the SD-WAN rules, it will match the implicit rule. The algorithm used in this case is based on source and destination IP addresses, which ensures that all traffic from a specific source IP to a specific destination IP is sent through the same interface.

References:

  • Fortinet Administration Guide - Implicit Rule

Correct Answer:
A. All traffic from a source IP to a destination IP is sent to the same interface.

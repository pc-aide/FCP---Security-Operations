Question 5

Refer to the exhibits, which show the system performance output and the default configuration of high memory usage thresholds in a FortiGate.

Based on the system performance output, what can be the two possible outcomes? (Choose two.)




Explanation:

It configured "set memory-use-threshold-red 88", so when memory comes up to 88% of usage and the memory consumption is 90%, FortiGate will enter conserve mode.

For this reason, administrators cannot change the configuration since conserve mode does not allow it.

References:

  • Conserve Mode - Fortinet Documentation

Correct Answers:
B. FortiGate has entered conserve mode.
C. Administrators cannot change the configuration.

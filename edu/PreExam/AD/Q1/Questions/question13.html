Question 13

When FortiGate performs SSL/SSH full inspection, you can decide how it should react when it detects an invalid certificate. Which three actions are valid actions that FortiGate can perform when it detects an invalid certificate? (Choose three.)





Explanation:

The valid actions are "Allow & Warning", "Trust & Allow", and "Block". Allow is not a valid option, and the untrusted certificate warning makes option A correct.

References:

  • FortiProxy SSL Inspection - Fortinet Documentation
  • FortiOS 7.6 Administration Guide

Correct Answer:
A. Allow & Warning
B. Trust & Allow
E. Block

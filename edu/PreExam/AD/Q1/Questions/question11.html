Question 11

Refer to the exhibits.

Based on the configuration, what will happen to Apple FaceTime if there are only a few calls originating or incoming?




Explanation:

The first rule that matches is the Excessive-Bandwidth block filter, and FaceTime is within that filter, which is applied even if only a few calls are present. This is covered in page 259 of the study guide, where it states that override rules take precedence. Therefore, Apple FaceTime will be blocked based on the Excessive-Bandwidth filter configuration.

References:

  • Filters for Application Control Groups
  • FortiOS New Features Guide

Correct Answer:
D. Apple FaceTime will be blocked, based on the Excessive-Bandwidth filter configuration.
